# Proyecto Web: Cafetería Las Orquídeas

Este repositorio contiene el código fuente del sitio web para "Cafetería Las Orquídeas", un establecimiento ubicado en Estepona, Málaga.

## Descripción

El sitio web es una página estática multipágina diseñada para presentar la cafetería, sus servicios, menú, galería de imágenes e información de contacto. Está construido con HTML5, CSS3 y JavaScript, y utiliza el framework Bootstrap 5 para un diseño responsive.

## Características

- **Diseño Responsive**: Adaptable a dispositivos móviles, tablets y ordenadores de escritorio.
- **Multilenguaje**: Soporte para español (ES) e inglés (EN) mediante archivos JSON.
- **SEO Optimizado**: Incluye metaetiquetas, schema markup (JSON-LD), `sitemap.xml` y `robots.txt`.
- **Componentes Interactivos**: Carrusel de imágenes, mapa de ubicación (Leaflet.js), y banner de consentimiento de cookies.
- **Accesibilidad**: Uso de atributos ARIA y estructura semántica de HTML.

## Estructura del Proyecto

- `/`: Archivos HTML principales.
- `/css`: Hojas de estilo personalizadas (`style.css`).
- `/imagenes`: Recursos gráficos (logo, fotos, etc.).
- `/js`: Lógica de la aplicación (`main.js` para traducciones y cookies).
- `/locales`: Archivos de traducción (`es.json`, `en.json`).
